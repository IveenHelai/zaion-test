## Description
Zaion websocket technical test

## Cloner le repository
```bash
$ git clone https://github.com/IveenHelai/zaion-test.git

```
## Installation

```bash
$ cd zaion-test
$ npm i
```

## Démarrer l'application
```bash
$ npm run start

```


#### Le websocket est désormais accessible depuis l'adresse suivante : 
ws://localhost:3050
